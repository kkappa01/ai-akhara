## ğŸš€ Getting Started

### Prerequisites

- Node.js 18+ 
- npm, yarn, pnpm, or bun

### Installation

1. Clone the repository:
```bash
git clone https://github.com/your-username/ai-akhara.git
cd ai-akhara
```

2. Install dependencies:
```bash

# or

# or
```
